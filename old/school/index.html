HTML Glossar

Der DOCTYPE einer Website bestimmt, wie der Browser die Website interpretieren soll.

<!DOCTYPE HTML>

Der <head> einer Website ist der Kopf der Seite und in ihm sind grundlegende Informationen enthalten.

<head> Inhalt </head>

Der <body> einer Website ist der Körper. Er beinhaltet alle Informationen die der Nutzer sehen kann.

<body> Inhalt </body>

Ein <div> ist ein Container auf einer Website, welcher für gewöhnlich Elemente wie Texte und Bilder beinhaltet.

<div id="ID" class="Klasse"> Inhalt </div>

Ein <p> Tag beschreibt einen Textparagraphen - daher das "p".

<p id="ID" class="Klasse"> Text </p>

Der <li> ist ein sogenanntes "list item", also ein Eintrag in einer Liste.

<ul>
    <li> Listen Eintrag 1 </li>
    <li> Listen Eintrag 2 </li>
    <li> Listen Eintrag 3 </li>
</ul>

Das <img> ist ein image, also ein Bild.

<img src="../img/img.jpg"></img>
